Detailed Description

template<typename TInternalType, typename TExternalType>
class itk::Accessor::ComplexToPhasePixelAccessor< TInternalType, TExternalType >

Give access to the Phase part of a std::complex<> value.

ComplexToPhasePixelAccessor is templated over an internal type and an external type representation. The internal type is an std::complex<T> and the external part is a type T. This class casts the input, applies the function to it, and casts the result according to the types defined as template parameters.
