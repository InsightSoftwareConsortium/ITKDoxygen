Detailed Description

template<typename TImage>
class itk::ImageReverseIterator< TImage >

A multi-dimensional image iterator designed to walk a specified region in reverse.

This is a subclass of ImageReverseConstIterator that adds write-access functionality. Please see ImageReverseConstIterator for details.

MORE INFORMATION
For a complete description of the ITK Image Iterators and their API, please see the Iterators chapter in the ITK Software Guide. The ITK Software Guide is available in print and as a free .pdf download from https://www.itk.org.
