Detailed Description

template<typename TNodeType>
class itk::ConstSparseFieldLayerIterator< TNodeType >

Used to iterate through an itkSparseFieldLayer.

This class is modelled on the STL Forward Iterator concept, and is used to iterate through an itkSparseFieldLayer.
