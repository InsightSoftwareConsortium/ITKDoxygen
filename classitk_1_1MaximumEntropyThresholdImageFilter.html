Detailed Description

template<typename TInputImage, typename TOutputImage, typename TMaskImage = TOutputImage>
class itk::MaximumEntropyThresholdImageFilter< TInputImage, TOutputImage, TMaskImage >

Threshold an image using the MaximumEntropy Threshold.

This filter creates a binary thresholded image that separates an image into foreground and background components. The filter computes the threshold using the MaximumEntropyThresholdCalculator and applies that threshold to the input image using the BinaryThresholdImageFilter.
