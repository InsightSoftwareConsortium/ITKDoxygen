Detailed Description

template<typename TImage>
class itk::LevelSetDenseImage< TImage >

Base class for the "dense" representation of a level-set function on one image.

This representation is a "dense" level-set function, i.e. it defines a level-set function on a grid (more precisely the underlying structure is an Image).

Template Parameters
TImageInput image type of the level set function
