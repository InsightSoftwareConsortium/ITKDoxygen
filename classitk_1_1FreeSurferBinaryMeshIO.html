itk::FreeSurferBinaryMeshIO Class Reference
Data Processing Objects » Filters » Input and Output Filters | Group IO » Module ITKIOMeshFreeSurfer

#include <itkFreeSurferBinaryMeshIO.h>

Detailed Description

This class defines how to read Freesurfer binary surface file format. To use IO factory, define the suffix as *.fsb.
