Detailed Description

template<unsigned int ObjectDimension>
class itk::CastSpatialObjectFilter< ObjectDimension >

This filter casts one spatialobject to another, when the class hierarchy supports it (e.g., Tube to PointBased). Particularly useful in Python where casting objects without public constructors (e.g., objects managed by smartpointers) is problematic.
