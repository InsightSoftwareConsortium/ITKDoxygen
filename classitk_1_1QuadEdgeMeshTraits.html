Detailed Description

template<typename TPixel, unsigned int VPointDimension, typename TPData, typename TDData, typename TCoordinate = float, typename TInterpolationWeight = float>
class itk::QuadEdgeMeshTraits< TPixel, VPointDimension, TPData, TDData, TCoordinate, TInterpolationWeight >

Class holding the traits of the QuadEdgeMesh.

This class is a variant of the MeshTraits that adds the traits defined in the QuadEdgeMeshCellTraitsInfo class.

Author
Alexandre Gouaillard, Leonardo Florez-Valencia, Eric Boix

This implementation was contributed as a paper to the Insight Journal https://doi.org/10.54294/4mx7kk
