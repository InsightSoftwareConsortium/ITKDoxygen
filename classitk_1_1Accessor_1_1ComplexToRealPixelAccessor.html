Detailed Description

template<typename TInternalType, typename TExternalType>
class itk::Accessor::ComplexToRealPixelAccessor< TInternalType, TExternalType >

Give access to the Real part of a std::complex<> value.

ComplexToRealPixelAccessor is templated over an internal type and an external type representation. The internal type is an std::complex<T> and the external part is a type T. This class casts the input, applies the function to it, and casts the result according to the types defined as template parameters.
