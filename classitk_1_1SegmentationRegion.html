Detailed Description

Base class for SegmentationRegion object.

itkSegmentationRegion is the base class for the SegmentationRegion objects. It provides the basic function definitions that are inherent to a SegmentationRegion objects. A region object is defined by the label it owns. We use integer labels to represent a region. This object stores the region label. The user can get the area and a label associated with the region through access functions provided publicly. This class provides an interface method called ApplySegmentationRegion is defined as virtual to enable future extension of the region object. This object can be used where region in images needs to be defined. Child classes can be derived from this class for tailoring them towards specific algorithmic needs.
