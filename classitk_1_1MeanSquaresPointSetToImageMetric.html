Detailed Description

template<typename TFixedPointSet, typename TMovingImage>
class itk::MeanSquaresPointSetToImageMetric< TFixedPointSet, TMovingImage >

Computes similarity between pixel values of a point set and intensity values of an image.

This metric computes the average squared differences between pixels in the point set and the transformed set of pixels.

Spatial correspondence between both images is established through a Transform.
